# Mapas das Regiões - Inteligência Global

Este diretório contém os mapas SVG das regiões utilizados na página de Inteligência Global.

## Arquivos Necessários

Adicione os seguintes arquivos SVG neste diretório:

1. **americas.svg** - Mapa das Américas (América do Norte, Central e do Sul)
2. **europe.svg** - Mapa da Europa
3. **asia.svg** - Mapa da Ásia
4. **africa.svg** - Mapa da África
5. **oceania.svg** - Mapa da Oceania (Austrália, Nova Zelândia, ilhas do Pacífico)

## Especificações Técnicas

- **Formato**: SVG (Scalable Vector Graphics)
- **Tamanho recomendado**: 120px de largura (altura proporcional)
- **Estilo**: Preferencialmente em estilo minimalista, contorno simples
- **Cores**: Os mapas serão exibidos com filtros CSS, então cores neutras funcionam melhor

## Onde Encontrar Mapas SVG

Você pode encontrar mapas SVG gratuitos em:
- [Simplemaps](https://simplemaps.com/resources/svg-maps)
- [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:SVG_maps)
- [Free SVG Maps](https://www.amcharts.com/svg-maps/)

## Fallback

Caso os arquivos SVG não estejam disponíveis, o sistema automaticamente exibirá ícones Font Awesome como fallback:
- Américas: `fa-globe-americas`
- Europa: `fa-globe-europe`
- Ásia: `fa-globe-asia`
- África: `fa-globe-africa`
- Oceania: `fa-globe-oceania`

## Nota

Os mapas são carregados via GitHub Raw URLs. Certifique-se de que os arquivos estejam commitados e disponíveis no repositório para que sejam exibidos corretamente.
